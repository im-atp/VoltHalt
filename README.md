# VoltHalt

An Android app that monitors your battery level and plays an alarm when it reaches your custom charge target — so you never overcharge again.

## Features

- Set a custom battery percentage target (e.g., 80%)
- Alarm plays at max volume when target is reached
- "Max Charge" screen auto-dismisses when charger is unplugged
- Persistent background monitoring service
- Clean Material 3 UI with Jetpack Compose
- Share the APK directly from the app
